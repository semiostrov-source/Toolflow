## Warehouses feature

The warehouses feature will represent storage locations in the system, including their structure, capacity, and high-level status. It will eventually include UI components and domain types for browsing warehouses, understanding where inventory is stored, and viewing basic operational information for each location.

Over time, this module will grow to support more detailed warehouse-centric workflows, such as zone-level views, capacity signals, and cross-linking with inventory and request flows, while keeping warehouse-specific concepts and behavior encapsulated within this feature boundary.
