## Inventory feature

The inventory feature will focus on managing items and their movements across the system, including stock levels, incoming and outgoing item flows, and basic tracking needed for the MVP. It will eventually contain UI components, domain types, and utilities related to viewing, filtering, and understanding inventory at a glance.

Over time, this module will evolve to support more advanced scenarios such as movement history, reconciliation workflows, and integrations with warehouse and request flows, while keeping all inventory-specific logic and presentation concerns encapsulated within this feature boundary.
