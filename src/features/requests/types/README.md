## Requests types

This directory will contain TypeScript types and interfaces for request-related concepts, such as request payloads, statuses, categories, and audit information. These types should capture the core data structures needed to represent and process requests in the system, independent of specific API or storage implementations.

Keep request-specific types here and reuse shared/types for cross-cutting primitives to maintain a clean separation of concerns between the requests feature and other parts of the application.
